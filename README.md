# Personal Information Manager

## Project Description

This is my first Python project. The program stores and displays personal information using variables, user input, string formatting, and basic validation.

## What I Learned

* Variables and Data Types
* Input and Output Operations
* String Formatting using f-strings
* Input Validation using loops
* String Methods such as strip() and title()

## Features

* Stores personal information (name, age, city, hobby)
* Accepts user input for favorite food and favorite color
* Validates empty inputs
* Calculates age in months
* Displays information in a formatted manner
* Uses comments for code documentation

## How to Run

1. Install Python
2. Open Terminal or Command Prompt
3. Navigate to the project folder

```bash
python personal_info.py
```

4. Enter the required information when prompted

## Challenges & Solutions

### Challenge

User might enter empty input.

### Solution

Used a while loop to repeatedly ask for input until a valid value is entered.

### Challenge

Formatting the output neatly.

### Solution

Used f-strings and separators for better readability.
